# 🌱 AgriAI – AI-Powered Farming Assistant

AgriAI is an AI-driven farming platform that empowers farmers with **crop prediction**, **disease detection**, **personalized recommendations**, and **weather forecasting** to improve yield and sustainability.

## 🚀 Overview
AgriAI is designed for **farmers, researchers, and agritech companies** to:
- Predict best crops for specific conditions.
- Detect plant diseases from images.
- Provide tailored advice using AI chatbot.
- Monitor weather conditions and receive alerts.

---

## ✨ Features
- 🌾 **Crop Recommendation** – AI suggests best crops based on soil & climate.
- 🦠 **Disease Detection** – Upload plant images for instant analysis.
- 🤖 **AI Chatbot** – Get instant farming advice 24/7.
- ☁ **Weather Alerts** – Stay updated with local forecasts.
- 📊 **Reports & Analytics** – Track progress and health trends.
- 📱 **Responsive Design** – Works on all devices.

---

## 🛠 Tech Stack
**Frontend:** React.js (Vite), Tailwind CSS, ShadCN/UI  
**Backend:** FastAPI, TensorFlow/PyTorch, OpenCV  
**Database:** SQLite / PostgreSQL  
**Others:** REST APIs, Weather API Integration

## ⚙ Setup & Installation

### 1️⃣ Clone Repo
```bash
git clone https://github.com/<your-username>/AgriAI.git
cd AgriAI

2️⃣ Backend Setup
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

3️⃣ Frontend Setup
cd frontend
npm install
npm run dev

▶ Usage

Open http://localhost:5173 in browser.

Use crop prediction form.

Upload plant images for analysis.

Chat with the AI assistant.
